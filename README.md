# Telegram Support Bot

Bot de automatización desarrollado en Python utilizando Aiogram.

## Funcionalidades
- Respuestas automáticas
- Flujos conversacionales
- Validación de datos
- Simulación de soporte técnico

## Tecnologías
- Python
- Aiogram
- Lógica de estados

## Objetivo
Automatizar consultas frecuentes y mejorar tiempos de respuesta en entornos de soporte.
